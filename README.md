# Student Placement Predictor

A Machine Learning web application built using Logistic Regression and Streamlit.

##  Features
- Predicts student placement based on IQ and CGPA
- Professional UI
- Real-time predictions
- Deployed online

##  Tech Stack
- Python
- Streamlit
- Scikit-learn
- NumPy
